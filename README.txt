

Overview

MostAnagramsFinder is a command-line Java application designed to find the word or words with the most anagrams from a given dictionary file. The program supports three data structures: Binary Search Tree (BST), Red-Black Tree (RBT), and Hash Map.

Usage

java MostAnagramsFinder <dictionary file> <bst|rbt|hash>

- <dictionary file>: Path to the text file containing words to be analyzed.
- <bst|rbt|hash>: Data structure to use for processing. 

Choose between:
- bst for Binary Search Tree
- rbt for Red-Black Tree
- hash for Hash Map

Example:

java MostAnagramsFinder dictionary.txt hash
